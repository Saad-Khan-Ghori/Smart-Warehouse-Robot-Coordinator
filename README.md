# Smart Warehouse Robot Coordinator

## Description
A multithreaded simulation of autonomous warehouse robots demonstrating synchronization using mutexes and semaphores.

## Features
- Robot threads (POSIX threads)
- Mutex-protected shared resources
- Semaphore-controlled critical zones
- Deadlock prevention
- Task scheduling system
- Performance analysis

## Technologies
- C
- POSIX Threads (pthread)
- Semaphores

make
./warehouse
